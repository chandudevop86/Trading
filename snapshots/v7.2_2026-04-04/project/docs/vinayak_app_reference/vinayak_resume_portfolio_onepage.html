PROJECT PORTFOLIO

Vinayak Trading Platform

API-first trading platform evolved from the original Trading project into a 3-tier architecture with live analysis, execution workflows, Redis caching, and report artifact storage.
Stack: FastAPI, Python, Redis, PostgreSQL, S3, AWS architecture, Telegram, Dhan integration

Highlights

  • Migrated a hybrid trading workspace into a cleaner 3-tier design.
  • Added live OHLCV service with Redis cache and file fallback.
  • Integrated live strategy analysis into a FastAPI dashboard.
  • Added report artifact generation with local and S3 storage paths.
  • Connected paper or live execution and Telegram summary support.
  • Prepared AWS-ready deployment direction with ALB, ECS, RDS, Redis, and S3.

Architecture Summary

Web Tier: Admin and workspace pages served through Nginx or ALB.
App Tier: FastAPI services for market data, strategy logic, execution, and notifications.
Data Tier: Relational DB, Redis hot-cache, and S3 or local report artifacts.
Production Goal: AWS-managed deployment with private app and data layers.
