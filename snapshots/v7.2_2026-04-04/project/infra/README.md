# Infrastructure Tier

This folder holds deployment, hosting, and operations assets for the separated Vinayak platform.

- `aws/`: EC2-first deployment scripts and related reference material
- `app/`: app-tier Docker, environment, and AWS packaging assets
- `cloudwatch/`: host and log metric shipping setup
- `config/`: shared environment examples
- `docker/`: legacy top-level Docker runtime
- `ec2/`: EC2 bootstrap and deployment scripts
- `k8s/`: Kubernetes manifests
- `scheduler/`: scheduled-task and reminder helpers
- `systemd/`: Linux service units
